Individual Reports

Server Memory Report

Full Life-Cycle Overview

Panel description

This report shows the development of the used, available and recommended Server Memory.
The Graphs display the entire life cycle from 2020-07-02 to 2026-07-23.

Value meaning

The purple line represents the currently available space, while the blue line represents the used space.
Therefore, the point where the lines cross is the date where the system will fail.
This assumes that the available space is not increased any further, since this requires a manual interaction.
The point of failure can be delayed by increasing the available space.
However, if the lines never cross, the space can be reduced accordingly.
A prediction function is used to forecast the trend for the systems remaining life cycle after the current date (2022-08-05).

The orange line represents the recommended space.
This is the space recommended according to the Blueprint vSnap Sizer sheets for the system to last until this day.
If the blue line is below this line, the system is developing slower than anticipated.
However, if the blue line is above the orange line, the system is developing quicker than anticipated.

One-Year Summary

Panel description

This report shows the anticipated free Server Memory and the expected usage percentage relative to the available memory.
All values are taken from the prediction graph, limited to the following year.
The Graphs display the range from now - 1 to now + 1 year.

Value meaning

A positive value means that space is still free after one year.
A negative value indicates the expected minimum value by which the capacity must be upgraded.
The percent values are displayed to allow an impression of the value to total ratio.

Setup-Check

Panel description

This report shows the currently existing Server Memory relative to the recommended Memory according to the Blueprint vSnap Sizer sheets at the end of the system lifetime.
All values are taken from the prediction graph of the system's entire life cycle.
The Graphs display the entire life cycle from 2020-07-02 to 2026-07-23.

Value meaning

A positive value means the system was set up with more space than recommended by the Sizer sheet.
A negative value indicates the difference required to reach the recommended size of the Sizer sheets.

A correct setup does not promise that the system will last the whole anticipated life-cycle.
Please check the used-data panels for such a forecast.

VADP Proxy Count Report

Full Life-Cycle Overview

Panel description

This report shows the development of the existing and recommended count of VADP Proxies.
The Graphs display the entire life cycle from 2020-07-02 to 2026-07-23.

Value meaning

The blue line represents the currently existing systems. The orange line represents the recommended count of systems.
This is the count recommended according to the Blueprint vSnap Sizer sheets for the SPP-System to last until its end of life cycle.
If the blue line is below this line, the system has too few Systems set up than recommended.
However, if the blue line is above the orange line, the system more Systems set up than recommend.
After the current date a static count of the currently existing Systems is assumed (2022-08-05).

One-Year Summary

Panel description
